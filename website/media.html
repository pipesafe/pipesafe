Title: Why PipeSafe?
Description: Type-safe MongoDB aggregation pipeline builder for TypeScript. Validates stages, field references, and operator types at compile time. Developed with input from the MongoDB Developer Experience team. Available on NPM.
PipeSafe
Type-safe MongoDB aggregation pipeline builder for TypeScript
>

Compile-Time Validation

Catch missing fields, type mismatches, and invalid operators before your code runs — not after deployment.

>

Type-Safe Field References

Field paths are validated against your document schema. Typos like "$totl" are caught instantly.

>

Automatic Output Inference

Each pipeline stage correctly types its output. InferOutputType gives you the final shape.

>

13 Pipeline Stages

$match, $project, $set, $group, $lookup, $unwind, $sort, $limit, $skip, $unset, $replaceRoot, $unionWith, $out.

>

DAG Orchestration

@pipesafe/manifold composes pipelines into typed DAGs with configurable materialization strategies.

>

MongoDB DevEx Input

Developed with input from the MongoDB Developer Experience team, with an open funding application to the NLNet NGI Zero Commons Fund.

PipeSafe
Two packages. One type-safe ecosystem.
@pipesafe/core
v0.6.0 — Published on NPM
Type-safe MongoDB aggregation pipeline builder. Validates everything at compile time.
  • 13 fully typed pipeline stages
  • Type-safe field path references
  • Expression operators ($concat, $multiply, etc.)
  • Collection-aware $lookup inference
  • Automatic output type inference
  • Flexible connection patterns
